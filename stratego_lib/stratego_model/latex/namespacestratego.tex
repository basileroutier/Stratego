\doxysubsection{Detailed Description}
The \mbox{\hyperlink{structstratego_1_1_terminal_color}{Terminal\+Color}} to write in color in terminal. So beautifull struct. 

This class represent the board. The board is a 2D array of \mbox{\hyperlink{classstratego_1_1_case}{Case}} and its size is 10x10. The board has also an attribute for the row and column, their value is 10 and can\textquotesingle{}t be modified.
